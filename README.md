# Project 124.

Написать веб-сервис с поддержкой мобильной верстки для получения ответов ChatGP. Особеностью платформы является несколько фактов:

- Можно задавать вопрос голосом
- Хранится полная история за все время общения 
- Можно задавать вопрос написанный текстом на фото

До регистрации пользователь может ознакомится с возможностями сервиса в виде лендинга. После регистрации пользователю доступно 5 операций в день. Предусмотреть 4 уровня подписки.

1) Бесплатная. Дает возможность использовать только картинку с текстом. Не более 50 в день.
2) 5$. Дает возможность использовать только картинку с текстом неорганиченное количество запросов
3) 25$. Как прошлая + просмотр истории испольльзования 
4) 50$. Как прошлая + возможность отправки голосовых сообщений не больше 15 секунд

Оплата - эмуляция с переходом на сайт любого банка.

Со стороны админа сервиса должна предоставлять возможность взаимодействия с базой клиентов и просмотром историей покупок.

Так же - возможность блокировки пользователя на время (1 день | 1 неделя | 3 месяца | 1 год | на всегда).

# Project 76.

Сделать сайт для работы со списком покупок
Сделать возможность для покупки
- Добавления
- Редактирования
- Удаления

Сама покупка должна иметь
- Заголовок
- Цену
- Фото
- Описание

Сделать отдельную страницу с выводом всех покупок и отдельно на добавление
На странице вывода всех сделать поиск (через JS)

Со стороны backEnd-a реализоватьт AspCore MVC
